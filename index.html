About JAMIFY

JAMIFY is a music streaming web application designed as part of our semester 3 mini-project. The goal of this project is to replicate a simplified version of popular music streaming platforms like Spotify.

Project Features:

  • Play, pause, and skip through a list of preloaded songs.
  • Interactive user interface with song titles and playback controls.
  • Responsive design for both desktop and mobile devices.
  • Progress bar to indicate song playback status.
  • Smooth and engaging user experience using JavaScript for song control.

Technologies Used:

  • HTML - For structuring the web page layout.
  • CSS - For designing a visually appealing user interface.
  • JavaScript - For dynamic song controls and interactions.

Goal:

The primary objective of this project is to deepen our understanding of front-end web development using core technologies. We aimed to create an interactive and responsive music player that mimics a real-world application, demonstrating practical coding skills in HTML, CSS, and JavaScript.
